# 오늘마무리

## 할 일

사용자가 `오늘은 Day X입니다. /오늘마무리`라고 말하면 그날 수업일지를 만든다.

## 파일 저장

가능하면 `내AI공부/수업일지/day-XX.md` 파일로 저장한다.

파일을 직접 만들 수 없으면 실패하지 말고, 사용자가 복사할 수 있게 아래 마크다운만 보여준다.

```markdown
# Day XX 수업일지

## 오늘 한 일
-

## 잘 통한 부탁 방식
-

## 막힌 점
-

## 내일 이어할 말
-
```

## 규칙

- Day 번호는 사용자가 준 문장에서 가져온다.
- 모르면 물어본다.
- 어려운 말은 쓰지 않는다.
- 파일 저장 실패를 길게 설명하지 않는다.
- 같은 Day를 다시 실행하면 같은 파일을 덮어써도 된다고 안내한다.
