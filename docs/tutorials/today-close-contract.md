# 매일 닫기 계약

## 목표

수강생이 하루 수업을 끝낼 때, 지금 대화 안에 "오늘 만든 것"과 "다음에 이어 할 말"이 남아야 한다.

## 입력

```text
오늘은 Day 7입니다.

오늘 만든 것을 짧게 정리해 주세요.

형식:
- 오늘 만든 것:
- 다음에 이어 할 말:

파일 저장이 꼭 필요한 결과물은 앞 단계 안내를 따릅니다. 오늘 대화에서 확인되지 않는 내용은 지어내지 말고 "확인 필요"라고 표시해 주세요.
```

Day 번호는 위저드가 문장 안에 넣어준다. AI가 파일을 스캔해서 추측하지 않는다.

## 출력 형식

```markdown
- 오늘 만든 것:
- 다음에 이어 할 말:
```

## 파일 저장

매일 닫기 단계에서는 파일 저장을 요구하지 않는다.

- 인보이스, 장부, 웹페이지 링크처럼 실제 파일이 필요한 결과물은 해당 본문 단계에서 저장을 안내한다.
- `ai-study/outputs`는 히스토리 접근이 안 될 때 쓰는 보조 증거다.
- 파일이 없어도 하루 완료는 실패가 아니다.

## 주간 회수

D5, D10, D15, D20에는 Claude/Codex 앱의 최근 대화와 작업 히스토리를 먼저 확인한다.

- 히스토리 접근이 가능하면 히스토리에서 패턴과 결과물을 뽑는다.
- 접근이 안 되면 `ai-study/outputs`와 현재 대화에서 확인 가능한 내용만 쓴다.
- 비밀번호, 토큰, 결제정보, 개인정보는 후보에 넣지 않는다.
- AGENTS.md에는 수강생이 승인한 후보만 반영한다.

## 출시 판단

자동 파일 저장은 필수 기능이 아니다. 수업의 필수 경로는 "대화 히스토리에 남기고, 주간에 회수하기"다.
