# /오늘마무리 계약

## 목표

수강생이 `/오늘마무리`를 입력하면 그날 수업 기록이 남아야 한다.

## 입력

```text
오늘은 Day 7입니다. /오늘마무리
```

Day 번호는 위저드가 문장 안에 넣어준다. AI가 파일을 스캔해서 추측하지 않는다.

## 출력 파일

- 폴더: `내AI공부/수업일지`
- 파일명: `day-XX.md`
- 같은 Day를 다시 실행하면 덮어써도 된다.

## 마크다운 형식

```markdown
# Day XX 수업일지

## 오늘 한 일
-

## 잘 통한 부탁 방식
-

## 막힌 점
-

## 내일 이어할 말
-
```

## 실패 시 fallback

앱이 파일을 직접 만들 수 없으면 실패로 치지 않는다. AI는 같은 내용을 마크다운으로 보여주고, 수강생은 `내AI공부/수업일지/day-XX.md`에 직접 붙여넣는다.

## 출시 판단

자동 파일 저장은 보너스다. 수업의 필수 경로는 “복사용 마크다운 생성”이다.
