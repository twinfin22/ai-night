# /오늘마무리 수동 QA

## 공통 준비

1. `내AI공부/수업일지` 폴더를 만든다.
2. 새 대화를 연다.
3. 아래 문장을 붙여넣는다.

```text
앞으로 제가 “/오늘마무리”라고 말하면, 오늘 배운 것과 잘 통한 부탁 방식을 수업일지 형식으로 정리해 주세요. 파일을 직접 만들 수 있으면 내AI공부/수업일지/day-XX.md로 저장하고, 파일을 만들 수 없으면 제가 복사할 수 있게 마크다운만 보여주세요.
```

## 테스트 1: Day 번호 유지

입력:

```text
오늘은 Day 7입니다. /오늘마무리
```

통과:

- 제목이 `Day 7 수업일지`다.
- 파일명 또는 안내 파일명이 `day-07.md`다.

## 테스트 2: 파일 저장 실패

입력:

```text
파일을 직접 만들 수 없다고 가정하고 /오늘마무리를 실행해 주세요.
```

통과:

- AI가 실패를 길게 변명하지 않는다.
- 바로 복사용 마크다운을 보여준다.

## 테스트 3: 같은 Day 재실행

입력:

```text
오늘은 Day 7입니다. 방금 내용을 조금 고쳐서 /오늘마무리 다시 실행해 주세요.
```

통과:

- 같은 `day-07.md`를 덮어써도 된다고 안내한다.
- 새 파일명으로 흩어지지 않는다.

## 테스트 4: 수업일지 회수

입력:

```text
Day 3부터 Day 11까지 수업일지가 있다고 가정하고, 나에 대한 AGENTS.md 초안을 만들어 주세요.
```

통과:

- 오늘 한 일, 잘 통한 부탁 방식, 막힌 점을 재료로 쓴다.
- 개인정보를 추측해서 만들지 않는다.
