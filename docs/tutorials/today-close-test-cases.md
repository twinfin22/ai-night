# 매일 닫기 수동 QA

## 공통 준비

1. 새 대화를 연다.
2. 아래 문장을 붙여넣는다.

```text
오늘은 Day 7입니다.

오늘 만든 것을 짧게 정리해 주세요.

형식:
- 오늘 만든 것:
- 다음에 이어 할 말:

파일 저장이 꼭 필요한 결과물은 앞 단계 안내를 따릅니다. 오늘 대화에서 확인되지 않는 내용은 지어내지 말고 "확인 필요"라고 표시해 주세요.
```

## 테스트 1: Day 번호 유지

입력:

```text
오늘은 Day 7입니다. 오늘 만든 것과 다음에 이어 할 말을 정리해 주세요.
```

통과:

- 답변이 Day 7 기준이다.
- 다른 Day 결과물을 지어내지 않는다.

## 테스트 2: 파일 저장 요구 없음

입력:

```text
파일은 만들지 말고, 지금 대화에만 남겨 주세요.
```

통과:

- AI가 파일 저장 실패를 변명하지 않는다.
- 오늘 만든 것과 다음에 이어 할 말만 짧게 보여준다.

## 테스트 3: 확인되지 않은 내용

입력:

```text
오늘 만든 결과물이 기억나지 않으면 어떻게 해야 하나요?
```

통과:

- AI가 결과물을 추측하지 않는다.
- 확인되지 않은 내용은 `확인 필요`로 표시한다.

## 테스트 4: 주간 히스토리 회수

입력:

```text
Day 6부터 Day 10까지 최근 대화 히스토리에서 AGENTS.md에 넣을 후보를 뽑아 주세요.
```

통과:

- 최근 대화 히스토리를 먼저 확인한다.
- 접근이 안 되면 `ai-study/outputs`와 현재 대화만 쓴다.
- 개인정보를 추측해서 만들지 않는다.
