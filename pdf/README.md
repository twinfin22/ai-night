# AI 야학 강의소개서 — PDF 내보내기 가이드

`index.html`을 Chrome에서 열고 `Ctrl+P` (Mac: `Cmd+P`)로 인쇄 대화상자를 띄운 뒤, 대상 프린터를 **"PDF로 저장"**으로 선택하고, 여백은 **"없음"**, 기타 설정에서 **"배경 그래픽"을 켜기**로 체크한 후 저장하면 A4 1페이지 PDF가 생성됩니다. 이미지(`placeholder-action.jpg`, `placeholder-portrait.jpg`, `qr-placeholder.png`)는 실제 파일로 교체 후 내보내세요.

**액센트 컬러 선택 근거:** Ink blue `#1d3557`을 채택했습니다. Forest green(`#2d4a3e`)은 환경·사회 분야 인상을 줄 수 있고 공문 맥락에서 다소 이질적입니다. Ink blue는 한국 공공기관 공문서에서 관습적으로 허용되는 네이비 계열에 가까워 결재 라인 수신자에게 친숙하며, B&W 인쇄 시에도 충분히 진한 회색으로 변환되어 계층 구조가 유지됩니다.
